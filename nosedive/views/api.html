Boostβ API Reference

/beta/api

Retrieve the documentation for the Boostβ API.

  • Request Type: GET
  • Arguments:
    • None
/beta/signup

Bring the user to the Prime Influencers Programme signup page.

  • Request Type: GET
  • Arguments:
    • None
/beta/signup

Register a user for priority access to the Prime Influencers Programme.

  • Request Type: POST
  • Arguments:
    • username: The user's username.
/beta/boost_rating

(support access only) Artificially boost a user's rating. Only use if granted explicit permission by authorities.

  • Request Type: POST
  • Arguments:
    • username: The user's username.
